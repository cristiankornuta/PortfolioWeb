Desarrollador de aplicaciones | Business Analyst | Python | Power BI | SQL | Machine Learning

Tipo: Freelance.

2013 – actualidad

Desempeño tareas freelance en las áreas de desarrollo de aplicaciones, análisis de negocios y soluciones tecnológicas utilizando Python, Java, .Net, Power BI, SQL y Machine Learning. Trabajando de manera independiente o en colaboración con equipos multidisciplinarios para proporcionar soluciones eficientes a clientes de diversos sectores.

  • Desarrollo de aplicaciones: Creación de aplicaciones personalizadas, desde el análisis de requerimientos hasta la implementación y mantenimiento, utilizando tecnologías como .Net. Java, Python, Django y frameworks modernos.
  • Business Analysis: Realización de estudios de viabilidad y análisis de procesos de negocio para optimizar y automatizar flujos de trabajo. Definición de soluciones tecnológicas a medida para mejorar la eficiencia operativa de las empresas.
  • Implementación de soluciones con Power BI: Diseño e implementación de dashboards interactivos y reportes dinámicos que facilitan la toma de decisiones estratégicas, utilizando Power BI para visualización y análisis de datos.
  • Desarrollo de modelos de Machine Learning: Creación de modelos predictivos y de clasificación utilizando Python y librerías como scikit-learn, TensorFlow y Keras, para ofrecer soluciones de análisis de datos avanzados y automatización de procesos.
  • Consultoría en SQL y bases de datos: Gestión y optimización de bases de datos, desarrollo de consultas SQL complejas y diseño de arquitecturas de bases de datos para mejorar el rendimiento y la accesibilidad de los datos en proyectos empresariales.
